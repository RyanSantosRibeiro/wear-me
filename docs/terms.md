## 2. Descrição do Serviço

O **WearMe** é uma plataforma SaaS (Software as a Service) que fornece uma API de provador virtual baseada em Inteligência Artificial. O serviço permite que lojistas integrem funcionalidades de visualização de produtos em seus clientes finais através de um widget JavaScript.

### 2.1. Funcionalidades Principais

- Geração de imagens de provador virtual utilizando IA generativa
- Widget JavaScript para integração em sites de e-commerce
- Dashboard para gerenciamento de uso, configurações e logs
- Sistema de planos com diferentes limites de requisições mensais
